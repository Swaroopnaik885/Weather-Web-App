# Weather Web App

A modern and responsive weather application that provides real-time weather information, hourly forecasts, and weekly outlooks using the Visual Crossing Weather API.

## Live Demo

https://swaroopnaik885.github.io/Weather-Web-App/

---

## Features

### Current Weather
- Real-time weather information
- Temperature
- Feels Like Temperature
- Wind Speed
- Humidity
- Rain Probability
- Weather Condition

### Forecasting
- Previous 24 Hours
- Next 24 Hours
- 7-Day Forecast

### Location Support
- Automatic geolocation detection
- Manual city search
- Global weather lookup

### Dynamic Experience
- Weather-based background images
- Smooth UI animations
- Responsive design for desktop and mobile devices

### User Experience
- Clean glassmorphism interface
- Dark theme design
- Fast loading experience
- Mobile-friendly layout
